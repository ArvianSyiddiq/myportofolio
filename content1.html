About this project

The personal finance management system is a collaborative group project developed using CodeIgniter 3, a PHP framework, and follows the MVC (Model-View-Controller) architecture. The application was built from scratch, with a focus on both frontend and backend development. Designed to help users efficiently manage their finances, the system provides tools to track income, expenses, and savings. It incorporates essential features such as a secure login and registration system, file upload functionality for storing financial documents, a calendar for managing deadlines and reminders, and interactive charts for visualizing financial trends. This project highlights strong teamwork, technical expertise, and a commitment to creating a user-friendly and secure application.
