About this project

The personal finance management system is a collaborative group project developed using CodeIgniter 3, a PHP framework, and follows the MVC (Model-View-Controller) architecture. The application was built from scratch, with a focus on both frontend and backend development. Designed to help users efficiently manage their finances, the system provides tools to track income, expenses, and savings. It incorporates essential features such as a secure login and registration system, file upload functionality for storing financial documents, a calendar for managing deadlines and reminders, and interactive charts for visualizing financial trends. This project highlights strong teamwork, technical expertise, and a commitment to creating a user-friendly and secure application.

Key Features:

  • User Authentication
    • Developed a login and registration system for secure user access.
  • File Upload
    • Added functionality for users to upload files (e.g., receipts, financial documents).
    • Ensured file validation and storage management.
  • Calendar Integration
    • Integrated a calendar feature to track financial events, deadlines, or reminders.
    • Enabled users to add, edit, and delete events.
  • Charts and Data Visualization
    • Implemented charts (e.g., bar charts, pie charts) to visualize financial data.
    • Used libraries or custom code to display income, expenses, and savings trends.
  • Frontend Development
    • Built the user interface (UI) from scratch.
    • Focused on responsive design for accessibility on different devices.
  • Backend Development
    • Developed the backend logic for handling user data, financial calculations, and database interactions.
    • Implemented CRUD (Create, Read, Update, Delete) operations for managing financial records.
  • Database Management
    • Designed and implemented a database schema to store user information, financial data, and uploaded files.
    • Used MySQL or another relational database system.
  • MVC Architecture
