About this project

This project performs hand gesture classification using Python with the TensorFlow library. It determines whether the hand gesture corresponds to rock, paper, or scissors, using 2000 hand photos and achieving an accuracy above 85%.

Key Metrics:

  • Split the dataset into 20% test data and 80% training data
  • Running 20 epoch iterations
  • Consistently achieves accuracy above 85%
